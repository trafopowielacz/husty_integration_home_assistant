## Zgłaszanie błędów

Przed utworzeniem zgłoszenia:

1. Sprawdź, czy używasz najnowszej wersji integracji.
2. Sprawdź dzienniki Home Assistanta.
3. Upewnij się, że klucz API jest aktywny.
4. Sprawdź, czy problem nie został już zgłoszony.

Nie publikuj:

- kluczy API,
- haseł,
- tokenów sesji,
- prywatnych adresów,
